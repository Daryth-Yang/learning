# 模块 2:AI 产品设计能力(4-5 周)

**周期**:4-5 周 | **目标**:从 UI 思维升级到 AI 产品思维,能独立设计 AI 产品

> 注:模块 2 详细内容在 Daryth 接近模块 1 完成时(W6-W7 期间)由教练扩写完整。这里先放骨架。

---

## 模块整体目标

- 能拆解任意一个 AI 产品的能力体系和 UX 决策
- 能给一个新需求设计完整 AI 产品方案(能力 + 工作流 + 对话 + 评测)
- 至少 5 周作业入仓库

## 周次概览

| Week | 主题 | 周作业(待细化)|
|---|---|---|
| W1 | AI 产品设计原则:能力边界 / 可控性 / 信任建立 | 拆解 Claude / Cursor / 即梦 各 1 个,各 800 字 |
| W2 | Agent / 工作流设计 | 设计"AI 设计稿评审助手"workflow |
| W3 | 对话式 UX 设计(Daryth 的核心战场)| 设计 3 个对话流程图(C 端 + B 端) |
| W4 | 评测体系(产品视角)| 给 W2 的 Agent 写评测方案 |
| W5 | 整合 + 验收 | 给真实需求做完整 AI 产品设计方案 |

## 关键差异化(为什么 Daryth 在这里有优势)

- 她有 C 端 + B 端 + 官网 设计经验 → 对话 UX 是天然战场
- 她在 D5 有真实产品环境 → 拆解 Clawtics / AI Slides 比纯外部研究有优势
