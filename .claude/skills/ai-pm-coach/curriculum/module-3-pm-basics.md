# 模块 3:PM 通用能力(4-5 周)⚠️ Daryth 最大短板

**周期**:4-5 周 | **目标**:补齐 PM 必需的硬技能(需求/PRD/数据/商业/项目管理)

> 注:模块 3 详细内容在 Daryth 接近模块 2 完成时由教练扩写完整。这里先放骨架。

---

## 模块整体目标

- 能独立做用户访谈并写访谈报告
- 能写出格式规范、覆盖完整的 PRD
- 懂数据驱动决策的基础(关键指标 / A/B 测试 / 漏斗)
- 懂 AI 产品的商业模式(订阅 / Token 计费 / API / 解决方案)
- 能做项目排期和跨职能协作

## 周次概览

| Week | 主题 | 周作业(待细化)|
|---|---|---|
| W1 | 需求挖掘 + 用户访谈 | 给 3 个真人做访谈,写报告 |
| W2 | PRD 写作 | 给模块 2 的 AI 产品写完整 PRD |
| W3 | 数据分析基础 | 用公开数据集做分析 + 报告 |
| W4 | AI 商业化 + 项目管理 | 商业模式画布 + 项目排期 |
| W5 | 整合 + 验收 | 综合 case:从需求到上线完整方案 |

## 重点关注

- 这是 Daryth 短板最明显的模块,可能需要拉长到 5 周甚至 6 周
- PRD 写作贯穿整个学习,不只在 W2:每个模块都有 PRD 练习,W2 是集中训练 + 系统化
