# 案例:数字人 / 视频生成(D5 产品)

> 多模态生成式 AI 产品。

## 产品定位

数字人 + 视频生成,可能用于:
- 直播带货(虚拟主播)
- 企业宣传(口播视频自动生成)
- 教育培训(虚拟讲师)
- 营销内容(短视频自动化)

## 关键技术特征

- **多模态模型**:文本 → 视频 / TTS(语音合成)/ 唇形同步 / 形象生成
- **训练成本高**:数字人形象通常需要"克隆训练"(2-30 分钟素材)
- **推理成本高**:每分钟视频要消耗大量算力
- **质量评测难**:没有客观标准(动作自然?表情真实?)

## 教练用法

### 模块 1 W2(模型生态 + 多模态)⭐ 核心案例
**作业相关**:给数字人产品做模型选型时,涉及:
- 视频生成模型(可灵 / Sora / Stable Video)
- TTS 模型(豆包 / 火山 / ElevenLabs)
- 形象生成模型(SD / FLUX / 即梦)

### 模块 2 W3(对话式 UX)
拆解数字人产品的 UX:
- 用户怎么"创建"自己的数字人?
- 上传素材的引导流程
- 预览和迭代体验
- 失败兜底(生成失败怎么办)

### 模块 3 W3 + W4(数据 + 商业化)
- 数字人产品的关键指标(创建完成率 / 视频生成成功率 / 用户复购)
- 商业模式:订阅 / 按生成时长付费 / 企业版
- 成本结构:训练成本 + 推理成本

## 教练注意事项

数字人 / 视频生成是 D5 公司的核心业务之一,Daryth 应该:
- 主动跟数字人产品的 PM/工程师交流
- 把这块作为模块 4 独立 MVP 的可能方向之一
