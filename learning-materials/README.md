## 🎯 资料组织原则

1. **每个链接都验证过**——教练通过 web 搜索找到并确认存在,不编 URL
2. **标注 "已看 / 待看"**——读完后在条目后面打勾
3. **分类按"主题"而不是"周次"**——主题可以跨周复用
4. **优先中文,英文标注 [EN]**

## 📝 怎么用

- **学习中**:从对应主题的文件里点链接直接看
- **看完后**:回到这里把条目前的 ⬜ 改成 ✅,留一句话感受
- **发现新资料**:让教练帮你验证后加进来,不要直接保存未验证的链接
